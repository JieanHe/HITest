# HITest

#### Description
一个SDK库的集成测试工具， SDK开发人员只需将需要测试的接口封装成 8个i64输入返回一个i32的形式，放入一个动态库，然后在toml配置文件提供动态库的路径和所有导出函数的符号；在另一个toml文件写测试用例即可执行测试。
